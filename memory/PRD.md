# PRD - ЕЛЛ Запчастини

## Оригінальна задача
Сайт по продажу запчастин до техніки ЕЛЛ, прайс-лист від 500 до 30000 гривень, з реєстрацією, красивим оформленням.

## Вибір користувача
- Категорії: Двигуни, Трансмісія, Гідравліка, Електрика, Ходова частина
- Доставка: Нова Пошта (відділення, поштомат, кур'єр)
- Оплата: Plata by Mono, LiqPay, Platon (місце під інтеграцію)
- JWT-авторизація
- Адмін панель
- Пошук, фільтрація, обране, історія замовлень
- Українська мова
- Телефон підтримки: +380965674376

## User Personas
1. **Покупець** - фермер/механік, шукає запчастини для МТЗ, ЗІЛ, комбайнів
2. **Адміністратор** - керує товарами, замовленнями, статусами

- Frontend: React 18, Tailwind CSS, Shadcn/UI
- Backend: FastAPI, Motor (MongoDB async)
- Database: MongoDB
- Auth: JWT (bcrypt)
